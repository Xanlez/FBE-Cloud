# Презентация FBE cloud

Шаблон в том же дизайне, что и сайт (тёмная тема, акцент `#5b9fd4`, карточки, логотип FBE).

## Файлы

| Файл | Назначение |
|------|------------|
| **`fbe-cloud-presentation.pptx`** | Презентация PowerPoint (8 слайдов) |
| `build_presentation.py` | Скрипт пересборки `.pptx` |
| `fbe-cloud-presentation.html` | Веб-версия (опционально) |
| `presentation-theme.css` | Стили для HTML-версии |

## PowerPoint

Откройте `fbe-cloud-presentation.pptx` в PowerPoint или LibreOffice Impress.

Пересобрать после правок:

```powershell
python documents/build_presentation.py
```

## Копия в «Документы» Windows

`%USERPROFILE%\Documents\FBE-cloud-presentation\fbe-cloud-presentation.pptx`
